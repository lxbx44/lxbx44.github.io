Arifetch: A simple system fetcher built in Rust

Welcome to Arifetch! A simple system fetch tool developed in Rust. With Arifetch, you can easily retrieve various system information and display it in a formatted table. Let's explore how you can install and use Arifetch

Installation

To get started with Arifetch, follow the installation instructions below:

Requirements

  • Rust
  • Cargo

Install Rust and Cargo

Before installing Arifetch, make sure you have Rust and Cargo installed on your system. Here's how to install them:

Linux / macOS:

curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh

source $HOME/.cargo/env

Windows:

Download the Rust installer from the official website. Follow the installation instructions to install Rust and Cargo.

Clone and Build Arifetch

  1. Clone the Arifetch repository:

git clone https://github.com/lxbx44/arifetch.git

cd arifetch/

  1. Build the project:

cargo build --release

Usage

To fetch system information using Arifetch, open your terminal and run the following command:

arifetch

This command will execute Arifetch and display a formatted table containing system information such as the operating system, kernel version, hostname, username, CPU details, and memory usage.

Manual Installation

If you prefer manual installation, you can download the files from the releases tab of the Arifetch repository.

How Does the Program Work?

Arifetch utilizes the Rust programming language and various libraries to fetch system information and generate a formatted table. Let's take a closer look at some key aspects of the program:

Fetching System Information

Arifetch uses the `sysinfo` and `whoami` crates to retrieve system details. These crates provide Rust bindings to system-level information and user information.

Formatting and Displaying Results

Arifetch uses the comfy_table crate to format and display the fetched system information in a table format. The `comfy_table` crate provides an easy-to-use API for creating tables with various formatting options.

Conclusion

Arifetch provides a simple and efficient way to fetch and display system information in a formatted table. By using Arifetch, you can quickly retrieve important system details and gain insights into your system's configuration. Give Arifetch a try and simplify the process of fetching system information!
