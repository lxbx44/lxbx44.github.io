PasswdManager

PasswdManager is a python application to simply store your passwords

Dependencies

  • Python3
  • Pip

Linux Installation

  1. $ git clone https://github.com/lxbx44/password_manager.git

  2. $ cd password_manager/

  3. $ pip install -r requirements.txt

  4. Additionally you can add the application in your path by running the following command:

    $ chmod +x main.py && sudo cp main.py /usr/bin/pmcli

PasswdManager is not aviable on windws yet

Usage

To use this program just run `pmcli` in your terminal
