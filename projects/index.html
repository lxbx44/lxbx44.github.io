PasswdManager

PasswdManager is a python application to simply and securely store your passwords

Dependencies

  • Python3
  • Pip

Linux Installation

  1. $ git clone https://github.com/lxbx44/password_manager.git

  2. $ cd password_manager/

  3. $ pip install -r requirements.txt

  4. Additionally, you can add the application in your path by running the following command:

    $ chmod +x main.py && sudo cp main.py /usr/bin/pmcli

PasswdManager is not available on Windows yet

Usage

To use this program, just run `pmcli` in your terminal

Flyser

Flyser is a simple and minimal file analyser made with rust to get basic information about any file

Dependencies

  • Rust
  • Cargo

Cargo Installation

  1. $ cargo install --git https://github.com/lxbx44/flyser.git

Usage

To use this program, just run the following in your terminal

$ flyser [file name]

or the following

$ flyser [path to file name]

For example: `flyser src/main.rs`
